# 介绍

waping UI 是一款基于 Vue 3 和 TypeScript 的 UI 组件库。

这款组件库是我为了更好的学习 Vue 3 和 TypeScript 而开发的，全程亲手编写，尽量不采用第三方库，包括你现在看到的这个官网也几乎都是我自己开发的。

所以不建议你将此 UI
库用于生产环境。源代码放在了
[https://github.com/wxyzcctv/waping-ui-1](#https://github.com/wxyzcctv/waping-ui-1)，历史提交信息规范，你可以按提交的顺序逐个查看；你也可以直接查看每个组件的源代码和示例。
